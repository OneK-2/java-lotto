## 로또

### 기능목록

1. 로또 번호 발행
   1. 구입금액에 해당하는 만큼 로또 발행
   2. 숫자 범위는 1~45
   3. 중복되지 않는 숫자 6개
2. 로또 구입 기능
   1. 1000원단위로 입력
   2. 1000원으로 나누어 떨어지지 않은 경우 예외처리
3. 로또 당첨 번호 생성 기능
   1. 당첨번호는 ,기준으로 구분
   2. 보너스 번호 입력
4. 발행한 로또 수량 및 번호 출력
   1. 오름차순 정렬
5. 번호 일치 확인
6. 당첨내역 출력
   1. 3~6개 각각 번호 일치 여부 
   2. 수익률은 소수점 둘째 자리 반올림
7. 예외상황 에러문구
